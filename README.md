# ODIN-PROJECT: LIBRARY

## Description:
This project is to practice DOM manipulation and object constructors using JavaScript. I was also able to add in my own features beyond the scope of the lesson such as adding a file input in the forms form uploading a cover image for a book.

## Skills used:
The skills used in making this project possible are:
- Basic HTML and CSS manipulation
- Proficiency in using CSS Grid and CSS Flexbox
- Basic JavaScript DOM Manipulation
- Basic JavaScript object constructors
